# NoteLock
A modern **Password Manager Web Application** built using **HTML, CSS, and JavaScript**. NoteLock allows users to securely store, organize, and manage their passwords in a clean, responsive interface. The application stores data locally using the browser's Local Storage, making it lightweight and easy to use without requiring a backend server

## ✨ Features
- 🔐 Add and save passwords
- 🔍 Search saved passwords
- 📋 Copy passwords to clipboard
- 👁️ Show / Hide passwords
- 🗑️ Delete passwords
- ✏️ Edit existing passwords
- 🎲 Random password generator
- 📊 Password strength indicator
- 🌙 Dark / Light mode
- 📱 Fully responsive design
- 💾 Local Storage support
- 🎨 Modern Glassmorphism UI
- ⚡ Smooth animations

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Google Fonts (Poppins)
- Browser Local Storage
